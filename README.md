# Fun Chat

WebSocket чат приложение для задания RSSchool.

## Установка и запуск

1. Установите зависимости:
\`\`\`bash
npm install
\`\`\`

2. Запустите сервер разработки:
\`\`\`bash
npm start
\`\`\`

3. Для production сборки:
\`\`\`bash
npm run build
\`\`\`

4. Деплой на GitHub Pages:
\`\`\`bash
npm run deploy
\`\`\`

## Сервер

Для работы приложения необходим сервер. Сервер запускается на порту 4000.

## Функционал

- Авторизация пользователей
- Обмен сообщениями в реальном времени
- Список пользователей онлайн
- Статусы сообщений (доставлено/прочитано)
- Удаление и редактирование сообщений
- Адаптивный дизайн

## Технологии

- TypeScript
- WebSocket
- Webpack
- SCSS
- ESLint + Prettier
